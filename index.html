About Me

Saya adalah siswi SMK jurusan Rekayasa Perangkat Lunak yang masih dalam tahap belajar pemrograman, khususnya pembuatan website. Saat ini saya sedang mempelajari dasar-dasar HTML dan CSS menggunakan Visual Studio Code sebagai media latihan.

Website ini dibuat sebagai latihan pembuatan portofolio dan surat lamaran kerja berbasis web sesuai dengan tugas sekolah. Melalui latihan ini, saya berharap dapat menambah pemahaman, pengalaman, serta meningkatkan kemampuan secara bertahap.

My Projects

Pada kelas XI semester 1, saya mengikuti proyek pembuatan web aplikasi hotel secara berkelompok. Saat itu saya belum memahami pembuatan aplikasi website, sehingga belum bisa banyak terlibat dalam proses pengerjaannya. Proyek ini menjadi pengalaman awal bagi saya untuk mengenal pembelajaran pembuatan website.

Saya membuat aplikasi laundry sederhana menggunakan Java dengan bantuan NetBeans pada kelas XII semester 1. Aplikasi ini dibuat dengan mengikuti tutorial YouTube yang diberikan sebagai bahan pembelajaran. Alhamdulillah, aplikasi tersebut dapat saya selesaikan dengan baik dan berjalan sesuai fungsi yang diharapkan, sehingga menambah pengalaman serta kepercayaan diri saya dalam membuat aplikasi berbasis Java.

Pada kelas XII semester 1, saya mengikuti proyek ASTS pembuatan aplikasi inventaris barang secara berkelompok. Dari tiga pilihan paket yang tersedia, yaitu website (Infinity), Java (NetBeans), dan Android, kelompok saya memilih Java menggunakan NetBeans. Alhamdulillah, saya dapat terlibat dalam proses pembuatan aplikasi bersama kelompok, meskipun aplikasi yang kami buat belum maksimal dan belum sepenuhnya tuntas.

Pada kelas XII semester 2, saya membuat aplikasi kalkulator menggunakan NetBeans. Aplikasi ini berhasil saya buat dengan bantuan tutorial YouTube, dan dari proses tersebut saya merasa bangga karena dapat menyelesaikan aplikasi sendiri sekaligus menambah pemahaman saya dalam pembuatan aplikasi.

Pada kelas XII semester 2, saya mempersiapkan diri untuk mengikuti UKOM 2026 jurusan Rekayasa Perangkat Lunak dengan membuat aplikasi aspirasi. Saya memilih paket Web menggunakan Visual Studio Code meskipun masih dalam tahap belajar, dan saya yakin dengan terus berusaha saya dapat menyelesaikannya dengan baik.
